# Linktree [PL]

Linktree to prosta strona internetowa stworzona w celu zgromadzenia wszystkich twoich linków do mediów społecznościowych w jednym miejscu. Można je łatwo udostępniać, a strona jest responsywna i estetycznie prezentuje się na różnych urządzeniach.
![](https://cdn.discordapp.com/attachments/838519186950586398/1086994602587869215/image.png)
## Funkcje

- Zgromadzenie wszystkich linków do mediów społecznościowych w jednym miejscu
- Responsywny i atrakcyjny wygląd
- Animacje i efekt paralaksy dla tła

## Jak zacząć?

1. Sklonuj to repozytorium.
2. Dostosuj linki do swoich mediów społecznościowych w pliku `index.html`.
3. Wgraj swoje zdjęcie profilowe i zaktualizuj link do niego w pliku `index.html`.
4. Wgraj strony na serwer lub użyj usługi hostingowej, takiej jak GitHub Pages, Netlify, Vercel itp.

# Linktree [EN]

Linktree is a simple web page created to gather all of your social media links in one place. It's easy to share and the page is responsive and visually appealing on various devices.

## Features

- Gather all your social media links in one place
- Responsive and visually appealing design
- Animations and parallax effect for the background

## Getting Started

1. Clone this repository.
2. Customize the social media links in the `index.html` file.
3. Upload your profile picture and update the link to it in the `index.html` file.
4. Upload the files to a server or use a hosting service like GitHub Pages, Netlify, Vercel, etc.
